# Privacy Policy — dubdub

_Last updated: 31 May 2026_

**dubdub does not collect any personal data.**

dubdub is a free, open-source app for browsing developer conferences and the
events around them. It has no user accounts, no analytics, no advertising, no
tracking, and no third-party SDKs.

## What the app stores

- **Favourites** are stored only on your device (in the app's local database).
  They are never uploaded, shared, or transmitted anywhere. Deleting the app
  removes them.
- **Display preferences** (such as showing past events) are stored only on your
  device.

## Network access

dubdub downloads its conference list as a public JSON file from this project's
GitHub repository (via jsDelivr, with `raw.githubusercontent.com` as a
fallback). These are ordinary, anonymous file downloads — the app sends no
account information, identifiers, or personal data with them. The list is also
bundled in the app so it works offline.

## Calendar

If you choose to add an event to your calendar, dubdub uses Apple's system
event editor. You confirm the save yourself, and you pick which calendar to use.
The app only writes the event you add — it never reads your calendar and never
transmits calendar data anywhere.

## Opening websites

Tapping a conference website, a map location, or the "Suggest a conference"
option opens that content using Apple's standard system components (in-app
Safari, Apple Maps, or the Mail composer). Anything you do there is governed by
Apple's and the destination site's own privacy practices, not dubdub's.

## Suggesting a conference

If you suggest a conference, the details you enter are placed into an email (or a
pre-filled GitHub issue) that **you** choose to send. dubdub does not store or
transmit this information on its own.

## Children's privacy

dubdub collects no data from anyone, including children.

## Changes to this policy

If this policy ever changes, the updated version will be posted here with a new
"Last updated" date.
